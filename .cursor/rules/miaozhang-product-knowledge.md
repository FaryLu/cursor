# 秒账基本介绍

秒账是一款进销存软件产品，覆盖多个行业，主要为商家提供下游客户管理、上游供应商管理、业务单据记录、产品库存管理、财务统计等综合功能，也对接了多种硬件，提供多种打印方式，配合商家开单，并且针对行业的特殊需求性提供一系列个性化的功能支持。秒账支持 PC 网页端、手机 App 和 Pad 使用。

同时秒账与秒账云仓（WMS 系统）打通，秒账商家在有产品交易往来时，可以将需要出入库的产品通过秒账一键同步到签约的云仓库内，库内收到秒账的数据后完成产品商家入库和拣货备货准备，省心快捷地完成产品销售出库、采购入库工作。

1. 用户可以通过手机号注册账号，注册之后就能登录使用。也可以通过微信、支付宝绑定账号后扫码快捷登录。
2. 登录后进入主页，左侧是功能菜单，右侧是对应菜单功能操作区，点击菜单进入使用。

# 客户模块

1. 维护下游客户信息，查看业务和财务往来，并通过数据面板展示合同金额、未收款、已收款等核心数据。
2. 大客户可指定业务员跟进。
3. 为保障客户之间每笔交易安全，提供实名认证渠道；商家与实名认证客户开单后可以签署电子合同，合同具有法律效益。
4. 供应商模块与客户模块类似，用于维护上游供应商信息、查看业务往来和相关数据统计。

# 销售单模块

1. 销售单是商家与客户之间交易的业务单据，也是秒账核心功能。
2. 秒账会根据不同行业提供不同定制功能、开单模式和交互方式。
3. 服装鞋服行业通常批量拿货出货，支持矩阵开单，同个产品可同时批量选择不同规格、颜色下单。
4. 布匹纺织行业支持裁剪售卖、缸号、匹号等特殊业务，并存在出库数量、打印标签数量、财务收款数量等多口径统计。
5. 对于有销售需求才采购的行业，支持销售开单同时完成采购。
6. 支持扫码开单、PDA 红外扫码开单、扫描产品条形码开单。
7. 支持多种打印方式，包括小票和 A4 格式，服装行业支持矩阵打印。
8. 支持线上线下两种收款方式，商家可生成二维码收款。
9. 业务单据支持生成电子合同，由商家和客户双方签署。

# 采购单

采购单是商家和上游供应商之间的业务单据，与销售单类似，会根据所属行业提供不同功能和开单模式。支持通过上游供应商出货单二维码一键扫码开单，提高采购入库效率。

# 退货单

退货单为客户、供应商提供销售产品退货和采购产品退货能力，财务上退款金额可冲销欠款。

# 加工单

1. 为有加工能力的用户提供简单加工工序功能。
2. 支持配置 BOM 表，配置后加工时可一键拆卸、一键组合、一键加工。
3. 支持按原材料成本 + 加工费计算成品成本。

# 产品

1. 管理 SKU，支持设置基本信息，如进价、售价。
2. 支持设置条形码，设置后可通过扫码开单，并适配 PDA 红外扫码。

# 库存

库存分为产品库存和云仓库存。

1. 产品库存支持查看商家自有库存，统计实际库存数量、可用库存和在途库存数量。
2. 云仓库存同步展示存放在云仓中的产品实时库存情况。

# 费用收入

秒账会统计全局财务情况。除销售单、采购单等业务单据外，也支持统计非业务类账单，例如水电费支出、员工福利发放等。

# 报表

1. 提供多张报表，从不同维度统计业务数据和财务数据，帮助用户分析运营情况。
2. 常用报表包括对账单，支持商户按月、季度、年与客户对账，也支持按产品明细、按业务汇总对账，并可将产品按矩阵形式合并对账。
3. 对账单支持导出、发邮件给下游客户、签署合同。
4. 还提供单品毛利、单据毛利、净利润等统计。

# 商户设置

商户设置包含多个子模块：

1. 基本信息：商户名称、Logo、地址等。
2. 行业信息：系统关键配置，支持配置行业和业务功能，影响全局功能和交互方式。
3. 业务属性：支持配置销售单快速采购、欠款口径、自定义公式、毛利算法等通用业务能力。
4. 货品属性：按行业配置产品属性，例如服装规格型号和颜色、外贸装箱外箱尺寸等。
5. 增值服务：包括分店模式、审批功能、希音对接服务等。
6. 打印信息：支持配置业务单据自定义打印信息，对账单也支持自定义格式打印和自定义模板。
7. 辅助资料：支持配置仓库、收款账户等基础资料。
8. 秒账云店：提供微信小程序线上商城，支持店铺基本信息和权限配置，客户下单后单据流转到秒账。
9. 线上支付：与第三方支付机构合作，支持商家申请线上收款。
10. 电子签章：与第三方合作，业务单据可生成电子合同，由商家与下游客户或供应商实名认证后签署。
11. 结账与归档：支持定期财务轧账，对历史封存不变的单据、产品、客户、供应商进行归档。
12. 员工管理：支持添加员工、分配角色和权限，保护商家数据。
13. 业务日志：以日志形式提供风险管控能力，并标记高危操作。

# WMS 相关

秒账与 WMS 实现对接，支持通过云仓入库单和云仓出库单向云仓发送入库和出库指令，出入库单据可与销售单、采购单等核心业务单据联动生成。当销售单确认送货信息时，系统可自动触发并生成对应云仓出库单，通知云仓拣选和出库。

对于出入库产品存在特殊处理需求的场景，如打印并粘贴唛头、物流标签或装箱作业，用户可在云仓出入库单中填写要求并上传附件，同步至云仓处理。云仓完成处理后，相关费用信息回传至秒账。

货主可通过秒账追踪云仓出入库单处理进度、查看云仓库存、动销天数，并使用云仓费用表、云仓库存分析表等报表分析仓储成本和库存动态。
