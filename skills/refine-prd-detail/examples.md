# 细化示例

以下示例用于说明：当标准 PRD 骨架已经确定后，如何继续细化到搜索区、列表区、新建页、编辑页、详情页，并明确字段权限、操作权限、数据范围。

## 示例输入
- 需求：销售订单功能
- 前提：已经完成影响模块分析，并已有标准结构 PRD 草稿
- 目标：把销售订单 PRD 细化到页面、字段、按钮、权限层

## 示例输出

```markdown
# 销售订单

1. 需求背景：
    - 销售订单用于前置承接销售计划、报价、客户确认等场景。
    - 用户可先录入订单，再根据订单生成或关联后续销售单，便于跟踪订单执行情况。
    - 本次需求重点是补齐销售订单的页面级、字段级、按钮级、权限级说明。
2. 影响模块：
    - 秒账：商户设置、销售订单、销售单、客户、产品、报表、打印、导入、导出、权限、业务日志

## 1 销售订单
### 1.1 列表页
#### 1.1 搜索区
1. 分店：
    - 交互逻辑：当商户设置中开启分店模式时显示该搜索项。
    - 字段权限：有查看销售订单列表权限时显示；无列表查看权限则整个页面不可见。
    - 数据范围：下拉框仅展示当前账号有权查看的数据范围内的分店；若有查看全部订单权限则展示全部有权分店，若仅可查看本人或本分店订单，则仅展示对应分店。
2. 客户：
    - 交互逻辑：客户名称模糊匹配。
    - 字段权限：有查看客户字段权限时显示该搜索项。
    - 数据范围：仅搜索当前账号有权限查看的订单数据中的客户。
3. 电话号码：
    - 交互逻辑：客户电话模糊匹配。
    - 字段权限：有查看客户电话字段权限时显示，否则隐藏。
    - 数据范围：仅在当前账号可见订单数据范围内匹配客户电话。
4. 单号：
    - 交互逻辑：订单单号模糊匹配。
    - 字段权限：有查看销售订单列表权限时显示。
    - 数据范围：仅搜索当前账号可见的销售订单数据。
5. 订单日期、账期：
    - 交互逻辑：支持切换，默认全部日期。
    - 字段权限：有查看销售订单列表权限时显示。
    - 数据范围：仅作用于当前账号可见的销售订单数据。
6. 产品：
    - 交互逻辑：产品名称模糊匹配。
    - 字段权限：有查看产品字段权限时显示。
    - 数据范围：仅匹配当前账号有权限查看的订单产品数据。
7. 规格型号：
    - 交互逻辑：当商户设置中开启规格型号时显示。
    - 字段权限：有查看规格型号字段权限时显示，否则隐藏。
    - 数据范围：仅匹配当前账号可见订单数据中的规格型号。
8. 颜色：
    - 交互逻辑：当商户设置中开启颜色时显示。
    - 字段权限：有查看颜色字段权限时显示，否则隐藏。
    - 数据范围：仅匹配当前账号可见订单数据中的颜色。
9. 订单状态：
    - 交互逻辑：下拉框，多选，选项为：未销售、部分销售、全部销售、超销售，默认不选中。
    - 字段权限：有查看销售订单列表权限时显示。
    - 数据范围：固定枚举值；筛选后仅返回当前账号有权查看的销售订单数据。

#### 1.2 列表字段
1. 单号：
    - 交互逻辑：显示销售订单号。
    - 字段权限：基础字段，默认固定展示；无详情查看权限时可展示但不可点击进入详情。
2. 分店名称：
    - 交互逻辑：开启分店时总店显示该列。
    - 字段权限：有查看分店字段权限时显示，否则隐藏。
3. 客户名称：
    - 交互逻辑：显示客户名称。
    - 字段权限：有查看客户字段权限时显示，否则隐藏。
4. 日期：
    - 交互逻辑：YY-MM-DD，取值为订单中的订单日期。
    - 字段权限：基础字段，默认固定展示。
5. 订单金额：
    - 交互逻辑：取销售订单详情中的订单金额。
    - 字段权限：有查看订单金额字段权限时显示，否则隐藏或空值，复用现有金额字段权限逻辑。
6. 销售金额：
    - 交互逻辑：订单中的销售金额 = 关联销售单金额，若为 0 显示 0。
    - 字段权限：有查看产品售价权限时显示，否则隐藏或空值。
7. 已收款：
    - 交互逻辑：订单中的已收款金额 = 关联销售单的已收款。
    - 字段权限：有查看收款金额字段权限时显示，否则隐藏或空值。
8. 订单状态：
    - 交互逻辑：
      - 未销售：全部产品行未选中或存在产品行选中且销售数量都为 0，红色显示。
      - 部分销售：存在任意产品行选中，且销售数量处于部分完成状态时，绿色显示。
      - 全部销售：全部产品行已选中且销售数量 = 总数量，黑色显示。
      - 超销售：存在任意产品行已选中且销售数量超过总数量时，额外显示标记。
    - 字段权限：基础字段，默认固定展示；若现有系统已有状态字段权限则复用现有规则。
9. 业务员：
    - 交互逻辑：单据中的业务员姓名。商户设置中开启开单选择业务员时显示该列。
    - 字段权限：有查看业务员字段权限时显示，否则隐藏。
10. 制单人：
    - 交互逻辑：制单员工姓名。
    - 字段权限：基础字段，默认固定展示。
11. 操作：
    - 交互逻辑：显示查看详情、编辑、删除、跳转关联销售单等操作入口。
    - 操作权限：有查看、编辑、删除、关联销售单查看等对应权限时显示和可点击；无权限时隐藏或点击报错，文案同现有。

#### 1.3 列表数据规则
1. 排序：
    - 默认按日期倒序排序，相同日期按创建时间倒序排序，排序规则同销售单。
    - 若支持列头排序，则单号、客户名称、日期、订单金额、销售金额、已收款、状态等字段按现有列表排序逻辑处理。
2. 数据权限：
    - 只能查看有查看销售订单权限的数据。
    - 若权限是“查看自己新建的销售订单”，则仅返回本人新建的数据。
    - 若权限是“查看全部销售订单”，则返回全部有权限范围内的数据。
    - 若同时受分店、客户、业务员等数据权限影响，则按现有销售单数据权限逻辑过滤。
3. 分页：
    - 默认 20 条/页。

#### 1.4 列表功能
1. 下载模板：
    - 操作权限：有下载模板或导入模板权限时显示，否则不显示。
    - 交互逻辑：销售订单导入模板中的字段说明复用销售单模板的通用规则，并按销售订单字段差异调整。
2. 导入：
    - 操作权限：有导入销售订单权限时显示，否则不显示。
    - 交互逻辑：销售订单导入校验与销售单相同字段校验逻辑一致；订单号、订单日期等字段按销售订单口径校验。
3. 批量导出：
    - 操作权限：有批量导出权限时显示，否则不显示。
    - 数据范围：仅可导出当前账号有权查看的数据；无字段权限的字段不导出或导出空值，复用现有导出逻辑。
4. 批量打印：
    - 操作权限：有批量打印权限时显示，否则不显示。
    - 交互逻辑：必须选中单据，否则提示选择需打印的单据；打印限制个数和打印样式复用现有单据打印规则。
5. 批量删除：
    - 操作权限：有批量删除权限且符合购买条件时显示，否则不显示。
    - 数据范围：仅可删除当前账号有删除权限的数据；无删除权限或不符合删除条件的数据自动过滤，提示文案同现有。
6. 新建销售订单：
    - 操作权限：账号有新建权限时显示，否则不显示。
    - 交互逻辑：点击进入新建销售订单页。

### 1.2 新建页
#### 1.1 基本信息区
1. 客户：
    - 是否必填：是
    - 输入格式：选择器 / 联想搜索
    - 默认值：空
    - 取值来源：客户资料
    - 字段权限：有新建销售订单权限时显示；若已有客户字段权限则复用现有规则。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：选择客户后可带出客户联系方式、账期、地址等默认信息。
    - 报错提示：未选择客户时，保存报错“请选择客户”。
    - 数据范围：仅展示当前账号有权查看和可选的客户数据。
2. 订单日期：
    - 是否必填：是
    - 输入格式：日期选择器
    - 默认值：当天
    - 取值来源：系统日期
    - 字段权限：基础字段，默认展示。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：用于后续列表展示、状态统计、报表取值。
    - 报错提示：为空时报错“请选择订单日期”。
3. 业务员：
    - 是否必填：按商户设置决定
    - 输入格式：下拉单选
    - 默认值：当前登录员工或老板
    - 取值来源：员工资料
    - 字段权限：当商户设置开启开单选择业务员时显示；若已有业务员字段权限则复用现有规则。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：影响单据归属和后续业务员统计报表。
    - 数据范围：仅展示当前账号有权选择的员工范围。
4. 备注：
- 是否必填：否
    - 输入格式：文本框
    - 默认值：空
    - 取值来源：用户输入
    - 字段权限：默认展示。
    - 编辑权限：有新建权限时可编辑。
    - 报错提示：超长时报错，文案同现有备注字段规则。

#### 1.2 产品明细区
1. 产品：
    - 是否必填：是
    - 输入格式：选择器 / 联想搜索
    - 默认值：空
    - 取值来源：产品资料
    - 字段权限：默认展示。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：选择产品后带出单位、规格、单价、库存相关信息。
    - 报错提示：未选择产品时保存报错“请选择产品”。
    - 数据范围：仅展示当前账号有权查看和可选的产品数据。
2. 规格型号 / 颜色：
    - 是否必填：按产品模式和商户设置决定
    - 输入格式：下拉或选择器
    - 默认值：空
    - 取值来源：产品维度资料
    - 字段权限：对应功能开启且有字段查看权限时显示。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：与产品联动刷新。
    - 数据范围：仅展示当前产品对应的可选规格和颜色。
3. 数量：
    - 是否必填：是
    - 输入格式：数字输入框
    - 默认值：空
    - 取值来源：用户输入
    - 字段权限：默认展示。
    - 编辑权限：有新建权限时可编辑。
    - 联动逻辑：与单价联动计算金额。
    - 报错提示：为空或非法格式时报错，文案同现有数量字段规则。
4. 单价：
    - 是否必填：按业务规则决定
    - 输入格式：数字输入框
    - 默认值：按客户价格或产品默认售价带出
    - 取值来源：产品价格体系或历史成交价
    - 字段权限：有查看产品售价权限时显示；无权限时隐藏或只读空值。
    - 编辑权限：有编辑价格类字段权限时可编辑，否则只读。
    - 联动逻辑：与数量联动计算金额。
    - 报错提示：输入非法格式时报错，文案同现有金额字段规则。
5. 金额：
    - 是否必填：否，计算字段
    - 输入格式：只读金额
    - 默认值：0
    - 取值来源：数量 * 单价
    - 字段权限：有查看订单金额权限时显示，否则隐藏或空值。
    - 编辑权限：不可编辑。
    - 联动逻辑：数量或单价变化时自动刷新。

#### 1.3 页面按钮
1. 保存：
    - 操作权限：有新建销售订单权限时显示，否则不显示。
    - 点击逻辑：校验主体信息和明细后保存单据。
    - 前置校验：客户、订单日期、产品明细、数量等必填项校验通过。
    - 成功提示：保存成功。
    - 失败提示：按字段报错并停留当前页面。
2. 取消：
    - 操作权限：进入页面即可使用。
    - 点击逻辑：返回列表页。
3. 新增明细行 / 删除明细行：
    - 操作权限：有新建销售订单权限时显示。
    - 点击逻辑：新增或删除当前明细行。

### 1.3 编辑页
#### 1.1 字段规则
1. 页面字段与新建页保持一致。
2. 字段展示规则与新建页一致。
3. 操作权限规则：
    - 有编辑权限时可进入编辑态。
    - 无编辑权限时详情可看，但编辑按钮不显示。

#### 1.2 页面按钮
1. 保存：
    - 操作权限：有编辑权限时显示。
    - 点击逻辑：保存修改内容，并刷新订单金额、状态等结果。
2. 删除：
    - 操作权限：有删除权限时显示。
    - 点击逻辑：删除销售订单并回退相关执行状态或关联关系。
3. 取消：
    - 操作权限：进入编辑页即可使用。
    - 点击逻辑：返回详情页。

### 1.4 详情页
#### 1.1 主体信息区
1. 客户、订单日期、状态、业务员、备注：
    - 展示逻辑：只读展示订单主体信息。
    - 取值来源：销售订单主表。
    - 字段权限：基础信息默认展示；业务员、客户电话等字段若有单独字段权限则复用现有规则。

#### 1.2 产品明细区
1. 产品、规格型号、颜色、数量：
    - 展示逻辑：只读展示订单明细信息。
    - 取值来源：销售订单明细。
    - 字段权限：默认展示；受商户设置控制的字段按现有逻辑处理。
2. 单价、金额、已收款、销售金额：
    - 展示逻辑：只读展示敏感金额信息。
    - 取值来源：销售订单明细及关联执行结果。
    - 字段权限：受金额类、价格类字段权限控制，无权限时隐藏或空值。

#### 1.3 页面按钮
1. 编辑：
    - 操作权限：有编辑权限时显示，否则不显示。
2. 删除：
    - 操作权限：有删除权限时显示，否则不显示。
3. 导出 / 打印：
    - 操作权限：有导出或打印权限时显示，否则不显示。
    - 数据范围：导出字段以详情页可见字段为准，无权限字段不导出或导出空值。
4. 生成销售单：
    - 操作权限：有销售单新建及来源单据引用权限时显示。
    - 点击逻辑：从当前销售订单带出可执行数据生成销售单。

## 2 关联模块
### 2.1 销售单
1. 销售订单可作为销售单的来源单据或关联单据。
2. 订单状态、销售数量、销售金额等信息需与销售单执行结果联动更新。

### 2.2 报表
1. 报表需补充销售订单相关统计口径，如订单金额、执行进度、未销售金额等。
2. 报表结果的数据范围受订单数据权限控制，敏感金额字段受字段权限控制。

### 2.3 打印 / 导入导出 / 权限 / 业务日志
1. 打印模板、导入模板、导出字段需补充销售订单对应字段。
2. 权限需覆盖列表查看、新建、编辑、删除、导入、导出、打印及敏感金额字段查看。
3. 业务日志需记录关键操作和关键字段变更。
```

## 补充片段：导入 / 导出 / 打印

以下片段用于说明：当需求已经明确支持 `导入`、`导出`、`打印` 时，不要只写“支持某能力”，而要继续拆成入口、字段范围、数据规则、权限规则。

```markdown
## 6 导入
### 6.1 列表页导入入口
#### 6.1 导入相关按钮
1. 下载模板：
   - 操作权限：有导入权限时显示，否则不显示。
   - 点击逻辑：点击后下载对应单据导入模板。
   - 模板字段范围：需覆盖主体字段和明细字段。
```
