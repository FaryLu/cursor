# 秒账 PRD 输出路径与命名约定

## 输出目录

| 产物 | 路径 | 命名规则 |
|------|------|----------|
| 影响模块 | `output/scope/` | `{需求名称}-影响模块.md` |
| 标准/细化 PRD | `output/prd/` | `{需求名称}.md` |

## 命名要求
- 取用户确认或提炼的**需求名称**，不含非法文件名字符。
- 细化版 PRD 覆盖同名 `{需求名称}.md`，不另建 `-细化版` 后缀文件，除非用户明确要求保留两个版本。

## 各步骤输出要求

### search-history-docs
- 必须写入 `output/scope/{需求名称}-影响模块.md`
- 同时向用户展示影响模块内容，等待确认后再进入下一步

### generate-final-prd
- 必须写入 `output/prd/{需求名称}.md`
- 产出文档档位：**标准骨架 PRD**（见全局规则文档档位定义）

### refine-prd-detail
- 必须写入 `output/prd/{需求名称}.md`（在骨架 PRD 基础上细化覆盖）
- 产出文档档位：**细化版 PRD**
