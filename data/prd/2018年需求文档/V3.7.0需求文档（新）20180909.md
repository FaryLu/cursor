# 51.收款状态和合同金额、送货金额正负方向不同，后台去掉限制

销售单：

按合同金额算：

|已收款+冲销欠款|=0或者≤|合同金额+费用垫付|的情况下

未收款=合同金额-已收金额-冲销欠款+费用垫付

|已收款+冲销欠款|≠0并且>|合同金额+费用垫付|的情况下

超收款=已收金额+冲销欠款-（合同金额+费用垫付），未收款=0

按送货金额算：

|已收金额+冲销欠款|=0或者≤|（送货金额+税额）*整单折扣-优惠金额+费用垫付|

未收款=（送货金额+税额）*整单折扣-优惠金额-已收金额-冲销欠款+费用垫付

|已收金额+冲销欠款|≠0并且>|（送货金额+税额）*整单折扣-优惠金额+费用垫付|

超收款=已收金额+冲销欠款-[（送货金额+税额）*整单折扣-优惠金额+费用垫付]

未收款状态：已收款+冲销欠款=0，合同金额+费用垫付≠0

部分收款状态：|已收款+冲销欠款|≠0<|合同金额+费用垫付|

全部收款状态：|已收款+冲销欠款|≠0=|合同金额+费用垫付|

超收款状态：|已收款+冲销欠款|≠0>|合同金额+费用垫付|

超收款状态下，未收款为0

按送货金额算欠款：

未收款状态：已收款+冲销欠款=0，(送货金额+税额)*整单折扣+费用垫付-优惠金额≠0

部分收款状态：|已收款+冲销欠款|≠0<|(送货金额+税额)*整单折扣+费用垫付-优惠金额|

全部收款状态：|已收款+冲销欠款|≠0=|(送货金额+税额)*整单折扣+费用垫付-优惠金额|

超收款状态：|已收款+冲销欠款|≠0>|(送货金额+税额)*整单折扣+费用垫付-优惠金额|

超收款状态下，未收款为0

采购单：

按合同金额算：

|已付款+冲销欠款|=0或者≤|合同金额-费用垫付|的情况下

未付款=合同金额-已付金额-冲销欠款-费用垫付

|已付款+冲销欠款|≠0并且>|合同金额-费用垫付|的情况下

超付款=已付金额+冲销欠款-（合同金额-费用垫付），未付款=0

按收货金额算：

|已付金额+冲销欠款|=0或者≤|(收货金额+税额)*整单折扣-优惠金额-费用垫付|

未付款=(收货金额+税额)*整单折扣-优惠金额-已付金额-冲销欠款-费用垫付

|已付金额+冲销欠款|≠0并且>|(收货金额+税额)*整单折扣-优惠金额-费用垫付|

超付款=已付金额+冲销欠款-[(收货金额+税额)*整单折扣-优惠金额-费用垫付]

按合同算欠款：

未付款状态：已付款+冲销欠款=0，合同金额-费用垫付≠0

部分付款状态：|已付款+冲销欠款|≠0<|合同金额-费用垫付|

全部付款状态：|已付款+冲销欠款|≠0=|合同金额-费用垫付|

超付款状态：|已付款+冲销欠款|≠0>|合同金额-费用垫付|

超付款状态下，未付款为0

按收货金额算欠款：

未付款状态：已付款+冲销欠款=0，(收货金额+税额)*整单折扣-费用垫付-优惠金额≠0

部分付款状态：|已付款+冲销欠款|≠0<|(收货金额+税额)*整单折扣-费用垫付-优惠金额|

全部付款状态：|已付款+冲销欠款|≠0=|(收货金额+税额)*整单折扣-费用垫付-优惠金额|
