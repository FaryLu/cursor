## 2. 全局补漏规则（跨系统）

| 触发条件 | 必须检查 | 说明 |

## CS-01 ERP 业务单据 → WMS 出/入库单

**何时检查**：改销售/采购/退货/调拨/加工/送货/收货；改云仓开单模式；改云仓仓或数量。

### 1.1 模式与触发时机（本地 `V4.9.0`）

| 模式 | 触发动作 | 生成 `[ERP]云仓出/入库单` | 送/收货单 |
|------|----------|---------------------------|-----------|
| 自动·流程一 | 保存业务单 / 确认送货收货 | 立即 | 生成 |
| 自动·流程二 | 云仓计划出/入库数量 > 0 | 按 planned qty | **不**生成送收货 |
| 手动 | 用户手工创建云仓单 | 手工时 | 视业务单 |

**跟随规则**：已生成单据跟随**保存时**商户设置；复制新增读**最新**商户设置。

### 1.2 单据映射表

| 源（ERP） | 目标 | 同步方向 | 证据 |
|-----------|------|----------|------|
| `[ERP]销售单`/送货 | `[ERP]云仓出库单` → `[WMS]出库单` | ERP→WMS | `V4.9.0` |
| `[ERP]采购单`/收货 | `[ERP]云仓入库单` → `[WMS]入库单` | ERP→WMS | `V4.9.0` |
| `[ERP]调拨单` | 云仓出+入库双单 | ERP→WMS | MCP v3.1.5-22 |
| `[ERP]加工单` | 云仓出/入库 | ERP→WMS | 同上 |
| `[ERP]采购退货单` | 云仓出库 | ERP→WMS | MCP v1.3.9-1 |
| `[ERP]商户设置` 云仓模式 | 上述生成时机 | BSS→ERP 初始默认 | `V4.9.0` |

### 1.3 接口失败

- ERP 调用 WMS 失败 → **终止**操作，提示「秒账云仓系统报错，请稍候再试，提示码：XXX」（`V5.4.0`）

## CS-08 账期 / 月账单欠款 → 三端锁定

**何时检查**：改账期、月账单审核/收款、解锁；任何可能触发出库的需求。

### 8.1 触发条件

> 月账单**已审核** + **未收款>0** + 当前时间 > 账单生成时间 + 账期天数

### 8.2 三端限制矩阵

| 限制动作 | ERP | WMS | 货主 App | 第三方 |
|----------|-----|-----|----------|--------|
| 新建/生成云仓出库 | ✗ | ✗ 新建/审核通过 | ✗ 新建出库 | ✗ 提交 |
| 查看库存 | — | — | ✗ | — |
| 自动审核出库 | — | **仍可通过** | — | — |
| 临时解锁 N 天 | 恢复 | 恢复 | 恢复 | 恢复 |

### 8.3 ERP 单据级拦截（摘要）

| 单据 | 自动模式 | 手动模式 |
|------|----------|----------|
| 销售单/采退 保存 | 云仓仓+送货>0 → 拦截 | 可先保存；手工生成云仓出库时拦 |
| 调拨/加工 保存 | 云仓仓+数量>0 → 拦截 | 同左 |
| 详情「生成云仓出库单」 | 拦截 | 拦截 |

**解锁权限**：运营销售(自己货主)、管理员、WMS 管理员/客服；短信验证（MCP v1.3.9-1）

**账期状态优先级**：临时解锁 > 已过账期 > 即将到期

## CS-10 计费链：WMS 作业 → 日/月账单

**何时检查**：改出入库操作费、工作量、加工费、优惠、日账单字段。

### 10.1 数据流

```
[WMS]出/入库单 收费明细/工作量
        ↓
[BSS/WMS]日账单（租金日账单 + 操作费日账单）
        ↓
[BSS]月账单（租金调整额 + 操作费 + 加工费 + 物流 + 其他）
        ↓ 审批后
[ERP]云仓账单 / 货主 App 支付
```

### 10.2 关键规则

| 规则 | 说明 | 证据 |
|------|------|------|
| 操作费拆分 | 出入库费拆分明细；货主签约配置 | MCP v2.2.7-7 |
| 优惠折上折 | 单箱体积、流转率、sku系数、出库8折可并存（规则见 v1.5.0-1） | MCP v1.5.0-1 |
| 日账单预估租金 | 生成日账单即展示预估；月底月账单补实际 | MCP v1.2.0-6 |
| 异形件 | 入库标记 → 租金/操作费公式 | MCP v1.5.5-4 |
| 工作量=0 拦截 | 见 CS-13 | MCP v1.6.0-3 |
| 财务报表应收 | 取月账单**折后价** | MCP v1.5.0-1 |

## CS-12 月账单重算时机

**何时检查**：改日账单、加工/出入库完结时间、手工结账、优惠档位变更。

| 月账单状态 | 日账单金额变更 | 行为 |
|------------|----------------|------|
| **未审批** | 变更 | 自动反映到月账单 |
| **已审批/已结算** | 变更 | **不**自动重算；须手工【重算】 |
| 优惠档位变更 | 流转率/sku 跨档 | 【重算优惠】（仅 BSS）；秒账/WMS 待重算后更新 |
| 财务结账 | 遗漏账单 | 须重算遗漏（MCP v1.6.5-1） |
| 预生成月账单 | 库存/单据变 | v1.7.8 起不实时更新，次日凌晨或手动刷新 |

**两种重算按钮**：
- 【重算】：日账单差额部分（原逻辑）
- 【重算优惠】：优惠档位变更 + 差额（MCP v1.5.0-1）；执行后隐藏【重算】

## CS-13 最后一票货与出清拦截

**何时检查**：出库将清空货主库存；完成出库；结束拣货；结束上架。

### 13.1 最后一票货流程（MCP v1.6.0-3）

| 步骤 | 系统 | 要点 |
|------|------|------|
| 1 标记 | WMS | 「完成出库可结账」；下单≥实际库存 |
| 2 结束拣货 | WMS | **先**手工结账弹框 → 再结束拣货 |
| 3 完成出库 | WMS | 货主将出清 → 检查变更未重算月账单 / 未收款 |
| 4 跳过 | WMS | 变更月账单可短信验证码跳过（特殊） |
| 5 收款 | BSS+WMS | 未付清 → 拦截完成出库 |

### 13.2 工作量费用拦截（与最后一票同属 v1.6.0-3）

- 自定义费用项：**有工作量但最终金额=0** → 禁止结束上架/结束拣货/完成出库
- 列表筛选项：「填写工作量但最终金额为0」

### 13.3 月账单变更提醒

- 变更月账单 → 消息通知有【重算】权限的销售/管理员
- 分享变更中账单 → 提示差额尚未重算
