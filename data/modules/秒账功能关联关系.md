# 秒账新功能影响模块模板

## 使用说明
- 本模板用于帮助 LLM 在生成需求文档前，补齐需求可能影响到的模块。
- 每个主功能按“功能定位 -> 常见直接影响 -> 常见间接影响 -> 横切检查项 -> 触发补漏规则 -> 可省略条件”的结构组织。
- 目标不是替代业务判断，而是让模型先覆盖高频关联，再按需求特征补查容易遗漏的模块。
- 后续可按模块逐步补充，不需要一次写全；若某项暂无明确内容，可先写 `待补充`。

## 通用问题

### 功能定位
- 所有功能需求默认都需要检查的共性问题。
- 常见上下游包括：UI 展示、性能、结账、归档、存储空间、时区、老数据兼容、新账号默认值、分店、渠道终端、扫码能力。

### 常见直接影响
- UI 新老模式：字段展示、样式、新老模式差异。
- 性能问题：压测数据、并发、多端操作、交互影响。
- 结账：已结账数据是否允许修改、如何拦截和提示。
- 归档：单据归档、时间归档、客户/供应商归档、产品归档。
- 存储空间：新增字段、模板、日志、附件是否增加存储占用。
- 时区：时间字段展示、筛选、排序、导入导出是否受影响。
- 分店：不同店铺之间的数据隔离、联动和权限差异。

### 常见间接影响
- 老数据兼容：历史数据是否兼容新规则。
- 新账号默认值：未配置场景下的默认行为。
- 秒账云店 / BSS / pad：若需求涉及渠道或终端，需要同步评估。
- 扫码能力：指尖扫码、扫条形码、扫 SN、扫布匹标签。

### 横切检查项
- 权限：查看、编辑、审核、删除、导出、打印等权限是否受影响。
- 打印：新增字段、规则、展示样式是否影响打印。
- 导入：模板字段、校验、错误提示是否同步。
- 导出：导出字段、导出顺序、导出权限是否同步。
- 报表：统计口径、筛选项、历史数据兼容是否受影响。
- 业务日志：关键字段、状态、金额变化是否需要记日志。
- 老数据兼容：旧数据展示和回显是否正常。
- 新账号默认值：默认开关、默认配置是否调整。
- 自定义模板：新增字段是否需要同步到模板能力。

### 触发补漏规则
- 涉及新增字段时，重点检查打印、导入、导出、自定义模板。
- 涉及金额、数量、库存、状态时，重点检查报表、权限、日志、上下游单据。
- 涉及商户设置开关时，重点检查老数据兼容和新账号默认值。
- 涉及扫码能力时，重点检查扫码类型和条码类能力。
- 涉及渠道终端时，重点检查云店、BSS、pad。

### 可省略条件
- 若仅调整纯文案或弱展示逻辑，且不影响字段、状态、金额、数量，可弱化报表、日志、消息提醒等检查，但建议说明原因。

## 销售单

### 功能定位
- 用于销售开单，并驱动出库、收款、利润统计等后续流程。
- 常见上下游包括：产品、库存、报表、打印、导入导出、商户设置、生成单据、消息提醒、权限、业务日志。

### 常见直接影响
- 销售单主流程能力：可负数、矩阵开单、传统弹框、批量开单。
- 销售单产品行能力：细码精细/非精细、装箱、赠品、副单位。
- 销售单规则能力：拒收、无编辑权限、平行多单位、审批模块、草稿单。
- 销售单展示能力：打印属性、多端操作。

### 常见间接影响
- 产品：产品属性、条码、单位、价格带出逻辑。
- 库存：数量、仓库、批次、SN、细码、可用库存、在途库存。
- 报表：销售流水、利润报表、产品销售总览、净利润、进销存、销售业绩。
- 商户设置：打印信息、多单位、平行多单位、SN/细码/批次/生产日期、业务属性。
- 复制新增：产品行复制、单据复制。
- 生成单据：销生采、快捷采购、销生退、销生加。
- 消息提醒：收款消息、送货消息、组合加工消息、分店消息。
- 业务日志：线上支付日志、关键字段变更日志。

### 横切检查项
- 权限：查看金额、编辑、审核、打印、导出等权限。
- 打印：A4、热敏、小助手、云打印、条码打印、打印属性。
- 导入：导入字段、模板、校验逻辑。
- 导出：列表导出、详情导出、汇总导出。
- 报表：利润、销售流水、业绩报表口径。
- 老数据兼容：老单据在新规则下的展示与计算。
- 新账号默认值：新账号默认是否开启相关能力。
- 自定义模板：新增字段或规则是否进入模板。

### 触发补漏规则
- 涉及新增字段时：必须检查打印、导入、导出、自定义模板。
- 涉及金额、折扣、优惠、欠款时：必须检查收款单、报表、权限、日志。
- 涉及数量、仓库、批次、SN、细码时：必须检查库存、送货单、销退单、报表。
- 涉及状态流转、审核、作废、草稿时：必须检查审批模块、消息提醒、业务日志、下游单据状态。
- 涉及扫码或条码时：必须检查扫码、产品、库存、打印。
- 涉及商户设置开关时：必须检查老数据、新账号、历史单据兼容。

### 可省略条件
- 若仅调整纯文案或弱展示，不影响字段、状态、金额、数量，可弱化库存、报表、消息提醒检查，但应说明原因。

- 业务日志：送货状态变更日志。

### 触发补漏规则

- 涉及来源单据时：必须检查销售单、生成单据、状态联动。

### 可省略条件
- 若仅调整送货文案或弱展示，不影响数量、出库和状态，可弱化库存与报表检查。

## 收货单

### 功能定位
- 用于承接采购到货和收货流程，关注入库出库明细、来源单据和数量回写。
- 常见上下游包括：采购单、库存、退货/调拨/加工相关单据、报表、消息提醒。

### 常见直接影响
- 收货单主流程能力：可负数。
- 收货单明细能力：出库明细、入库明细。

### 常见间接影响
- 采购单：来源单据、状态、数量联动。
- 销退单 / 采退单 / 调拨单 / 加工单：相关退货、调拨和加工联动。
- 库存：入库出库明细、库存数量变化。
- 报表：送收货明细表、进销存相关统计。
- 消息提醒：收货消息。

- 打印：收货单打印。
- 报表：送收货与库存统计。
- 业务日志：数量回写和状态联动日志。

- 涉及退货或调拨逻辑时：必须检查销退单、采退单、调拨单、加工单。

- 销售单：产品带出、价格、条码、单位、属性。
- 采购单：产品带出、价格、条码、单位、属性。
- 库存：产品库存、库存均价、可用库存、在途库存。
- 报表：产品销售总览、进销存、利润统计。
- 商户设置：行业属性、货品属性、多单位、平行多单位、SN/细码/批次/生产日期。
- 扫码：条形码、SN、布匹标签。
- 老数据：新增属性和禁用属性后的历史数据兼容。

### 横切检查项
- 权限：查看产品相关敏感字段、编辑、导入导出权限。
- 打印：产品相关打印字段、条码打印。
- 导入：产品导入模板、字段和校验。
- 导出：产品导出字段和模板。
- 报表：产品维度报表统计口径。
- 业务日志：产品字段变化是否需要留痕。
- 老数据兼容：老产品缺失新字段时的展示和默认值。
- 新账号默认值：相关产品属性能力的默认开关。
- 自定义模板：新增产品字段是否可用于模板。

### 触发补漏规则
- 涉及新增产品字段时：必须检查销售单、采购单、导入、导出、打印模板。
- 涉及条码、SN、细码、批次时：必须检查扫码、库存、打印、导入。
- 涉及单位、规格、颜色、多属性时：必须检查库存、报表、单据产品行展示。
- 涉及禁用属性或新增属性时：必须检查商户设置、老数据、新账号、历史单据兼容。

### 可省略条件
- 若只改产品列表查询交互，不影响字段、属性和带出逻辑，可弱化扫码与报表检查。

### 常见直接影响
- 客户/供应商字段。
- 归档、账期、欠款等基础业务能力。

### 常见间接影响
- 销售单 / 采购单：客户供应商带出和筛选。
- 收款单 / 付款单：往来对象和资金联动。
- 归档：归档后的历史单据展示和筛选。

### 横切检查项
- 权限：查看和编辑客户供应商权限。
- 报表：往来和账期统计。
- 老数据兼容：历史单据中的客户供应商回显。

### 触发补漏规则
- 涉及客户/供应商字段、归档、账期、欠款时：必须检查对应单据、报表、历史数据兼容。

### 可省略条件
- 若需求不涉及客户/供应商对象，可省略此项。
