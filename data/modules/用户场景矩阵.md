## 1. 使用说明

**匹配步骤**
1. 从需求提取**业务动作**（开单、送货、签约、拣货、重算…）
2. 判断是否**云仓货主**、**审批状态**、**账期状态**
3. 匹配场景 ID；读对应 CS 节 + `系统定位-三系统.md` 生命周期
4. 用「MCP 检索词」`get_requirement_detail` 补规则细节

**场景卡片字段说明**

| 字段 | 含义 |
|------|------|
| 前置条件 | 场景成立必要状态 |
| 主路径 | 用户操作链（→ 表示顺序） |
| 失败/分支 | 常见拦截与提示 |
| CS | `跨系统功能关联关系.md` 编号 |

## A. 进销存开单与云仓同步

### S02 销售送货触发云仓出库（自动·流程一）

| 项 | 内容 |
|----|------|
| **用户** | R02 开单员 → R06 操作员 |
| **前置** | 已签云仓；BSS 审批通过；商户设置=自动模式·流程一；未过账期 |
| **主路径** | 销售单保存/送货 → `[ERP]云仓出库单` → `[WMS]出库单` 待审核 → 拣货链 |
| **失败/分支** | 过账期 → 保存/送货拦截（CS-08）；WMS 接口失败 → ERP 报错终止 |
| **检索词** | 云仓出库单、自动模式、流程一 |
| **CS** | CS-01、CS-08 |
| **证据** | 本地 `V4.9.0` §流程一 |

### S03 采购收货入云仓

| 项 | 内容 |
|----|------|
| **用户** | R03 → R06 |
| **前置** | 云仓仓 + 自动/手动模式；货主有效 |
| **主路径** | 采购单/收货 → `[ERP]云仓入库单` → WMS 入库 → 上架 → 库存入账 |
| **失败/分支** | 货主要求有值 → 待审核入库详情弹框（MCP v3.0.1-26） |
| **CS** | CS-01、CS-05 |
| **证据** | 本地 `V4.9.0` |

### S04 手动创建云仓出入库单

| 项 | 内容 |
|----|------|
| **用户** | R02/R03 |
| **前置** | 商户设置=手动模式 |
| **主路径** | 业务单详情 → 手工创建云仓出/入库单 → 同步 WMS |
| **失败/分支** | 过账期 → 手工生成云仓出库单拦截；业务单可先保存 |
| **CS** | CS-01 |
| **证据** | 本地 `V4.9.0` §手动模式 |

### S05 云仓作业完成 → ERP 待确认
